简介

欢迎使用“AICC”的在线客服移动端开发者工具套件（SDK）。本文针对安卓端使用做详细说明，通过 SDK，可以在您的 APP 中快速集成访客端在线聊天的功能，以具备文本、图片、视频等类型消息收发。

环境要求

引入SDK

Maven 依赖方式

在项目根目录的 build.gradle 文件中添加如下配置

  allprojects {
    repositories {
      ...
      maven { url 'https://jitpack.io' }
    }
  }

引入开源工程oskit。 开源地址： link:chat-mobile/zip/online_sdk_1.7.20_kit_release.zip[下载]

在主 app 的 build.gradle 文件中添加如下依赖配置

  implementation 'com.github.ti-net:OnlineLib-Android:1.8.0'

混淆配置

-keep class com.tinet.** { *; }
-keep class org.tinet.** { *; }
-keep class com.org.** { *; }
-keep class com.lcw.** { *; }

权限适配

AndroidManifest.xml 中添加如下所需要的权限

<!--SD卡读写权限，发送文件、保存文件。-->
<uses-permission android:name="android.permission.WRITE_EXTERNAL_STORAGE"/>
<uses-permission android:name="android.permission.READ_EXTERNAL_STORAGE"/>
<!--录音权限，用于录音。-->
<uses-permission android:name="android.permission.RECORD_AUDIO"/>
<!--拍照权限，用于拍照（发送照片）-->
<uses-permission android:name="android.permission.CAMERA"/>


<!-- 安卓11+选择文件进行发送时查看所有文件必须权限  该权限属敏感权限，如有问题请咨询SDK技术支持 -->
<uses-permission android:name="android.permission.MANAGE_EXTERNAL_STORAGE" />
<uses-permission
    android:name="android.permission.MOUNT_UNMOUNT_FILESYSTEMS"
    tools:ignore="ProtectedPermissions" />

会话连接管理

功能描述

会话连接管理使用OnlineConnectManager类来实现，包括添加连接状态监听和移除连接状态监听（注：添加了连接状态监听在销毁context的时候需要移除连接监听，以免发生内存泄露）。监听回调使用OnlineConnectStatusListener。 OnlineConnectStatusListener可以对会话的连接中、连接成功、连接断开（调用disconnect方法或断线重连失败）、断线重连中、断线重连成功以及被踢下线等状态进行监听。

代码示例
xxxxxxxxxx

                    public void run() {

                    }
                });
            }

            public void onConnected() {

                    @Override

                    }
                });
            }

发送失败的消息管理

功能描述

实时消息发送失败会被OnlineMessageFailureManager进行管理。 开放了收集消息发送失败方法：messageSendFailure，移除指定发送失败的消息：reSendMessage，获取所有发送失败的消息：getAllFailureMessage，清空所有发送失败的消息。

代码示例
xxxxxxxxxx


//收集发送失败的消息
OnlineMessageFailureManager.getInstance().messageSendFailure(OnlineMessage message);


//移除指定发送失败的消息
OnlineMessageFailureManager.getInstance().reSendMessage(OnlineMessage message);


//获取所有发送失败的消息
OnlineMessageFailureManager.getInstance().getAllFailureMessage();


//清空所有发送失败的消息
OnlineMessageFailureManager.getInstance().getAllFailureMessage();

排队管理

功能描述

排队管理通过OnlineQueueManager类来实现，包括注册监听排队事件：registerQueueListener、退出排队监听：exitQueue、取消监听排队事件：unRegisterQueueListener以及触发排队监听方法：notifyQueueChanged。

代码示例
xxxxxxxxxx


//注册排队监听
OnlineQueueManager.registerQueueListener(OnlineQueueListener listener);


//触发排队监听
OnlineQueueManager.notifyQueueChanged(OnlineMessage message);


//退出排队监听
OnlineQueueManager.OnlineQueueManager.exitQueue();


//取消监听排队事件
OnlineQueueManager.unRegisterQueueListener();
